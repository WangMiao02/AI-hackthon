# 购票流程 — 卡片模板与交互细节

## 时段选择卡片（需预约时段的景区）

```
┌──────────────────────────────────────┐
│ 📅 选择入园时段                      │
│                                      │
│ 故宫博物院 · 5月25日（周六）         │
│                                      │
│ 🌅 上午场  08:30–12:00  余票 234 张  │
│ ☀️ 下午场  12:00–17:00  余票 89 张   │
│ 🎫 全天票  08:30–17:00  余票 156 张  │
│                                      │
│ [选上午场] [选下午场] [选全天票]     │
└──────────────────────────────────────┘
```

## 购票确认卡片

```
┌──────────────────────────────────────┐
│ 🎫 确认购票                          │
│                                      │
│ 🏛 故宫博物院                        │
│ 📅 5月25日（周六）上午场 08:30–12:00 │
│                                      │
│ 成人票 ¥60 × 2                       │
│ 💰 合计：¥120.00                     │
│                                      │
│ 👤 游客信息：                        │
│    张思怡 · 身份证 xxxxxx            │
│    李明   · 身份证 xxxxxx            │
│                                      │
│ 支付方式：美团支付（默认）           │
│                                      │
│      [确认购票]     [修改信息]       │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「确认购票」 | 调用 `order` skill 完成支付 → 购票结果处理 |
| 点击「修改信息」 | 返回修改游客信息或日期 |

## 购票成功卡片

```
┌──────────────────────────────────────┐
│ ✅ 购票成功！                        │
│                                      │
│ 🏛 故宫博物院                        │
│ 📅 5月25日（周六）上午场 08:30       │
│ 👤 张思怡、李明                      │
│ 💰 ¥120.00                           │
│                                      │
│ 📱 入园凭证：[查看二维码]            │
│                                      │
│ 📌 明天早上 7:30 我会提醒你出发      │
│ 🕙 建议提前 30 分钟到达              │
│                                      │
│ [查看电子票] [叫车去景区]            │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「查看电子票」 | 展示二维码/条形码 |
| 点击「叫车去景区」 | 触发 `ride_hailing`，目的地预填为景区地址 |

## 外地景区联动推荐

```
📌 这个景区在杭州，要帮你安排出行吗？

[帮我订票去杭州] [帮我订酒店] [不用了]
```

→ 点击「帮我订票去杭州」→ 触发 `trip_reserver`
→ 点击「帮我订酒店」→ 触发 `hotel_reserver`
