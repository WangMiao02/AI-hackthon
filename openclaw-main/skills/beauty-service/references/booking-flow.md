# 预约流程 — 卡片模板与交互细节

## 门店推荐卡片

```
┌──────────────────────────────────────┐
│ 💆 为你找到以下美容门店              │
│ 📍 基于你的位置：朝阳区XX路          │
│                                      │
│ 🥇 悦容·皮肤管理中心                │
│    ⭐ 4.9分 · 1286 人消费过          │
│    💰 人均 ¥268 · 距你 1.2km         │
│    🔁 你上次来过这家                 │
│                                      │
│ 🥈 BLOOM SPA                         │
│    ⭐ 4.8分 · 856 人消费过           │
│    💰 人均 ¥328 · 距你 2.1km         │
│    + 3家更多门店 ∨                   │
│                                      │
│ [选第一家]  [选第二家]  [自己看看]   │
└──────────────────────────────────────┘
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 🔁 你上次来过这家 | 历史有消费记录时显示，无则不显示 |
| 人均价格 | 取该品类的平均消费金额 |
| 自己看看 | 跳转美团丽人搜索页，Skill 结束 |

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「选第X家」 | 展示该店项目列表（见下） |
| 说"第一个"/"去悦容" | 识别后跳转项目列表 |
| 点击「自己看看」 | 跳转美团丽人搜索页，Skill 结束 |

**字段说明：**

| 字段 | 说明 |
|------|------|

| 操作 | 处理 |
|------|------|

| 点击「不指定，随机」 | 跳过技师选择，展示所有可用时段 |
