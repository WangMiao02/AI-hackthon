# 智能洞察与省钱建议 — 分析维度与卡片模板

## 触发条件

- 用户主动询问"帮我省钱""怎么减少开支""有什么优惠"等
- 月报生成时自动附带
- 超支预警时自动生成（加载 `references/alert-flow.md` 后联动）

## 洞察维度

| 分析维度 | 洞察示例 |
|---------|---------|
| 消费趋势 | "外卖消费连续3月上涨，月均增幅¥200" |
| 异常消费 | "本周打车消费是上周的2倍，有加班打车？" |
| 时段分析 | "夜宵外卖占外卖总消费25%，集中在23点后" |
| 优惠利用率 | "本月有3张未用红包已过期，损失约¥15" |
| 类目占比变化 | "娱乐门票消费占比从5%升至15%" |

## 省钱建议卡片

```
┌──────────────────────────────────────┐
│ 💡 为你找到 3 个省钱机会              │
│                                      │
│ 1️⃣ 外卖红包                          │
│    你有 2 张外卖满减券即将过期       │
│    预计可省：¥10                      │
│    [去领券]                           │
│                                      │
│ 2️⃣ 出行优化                          │
│    本周3次短途打车（<3km），         │
│    骑车可省约 ¥45                    │
│    [知道了]                           │
│                                      │
│ 3️⃣ 日用品凑单                        │
│    牙膏和洗衣液下月初到期需补，      │
│    一起买可享满减 ¥20                │
│    [提醒我]                           │
│                                      │
│ 📌 预计本月可省：¥75                 │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「去领券」 | 触发 `coupon_claim` skill |
| 点击「知道了」 | 标记该条建议已读，不重复推送 |
| 点击「提醒我」 | 设置补货提醒，联动 `supplies_reminder` |
