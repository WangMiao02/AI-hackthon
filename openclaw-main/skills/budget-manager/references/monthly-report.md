# 消费月报 — 卡片模板与趋势分析

## 触发条件

- 每月 1 号 9:00 自动推送上月月报（`trigger_type == "scheduled"`）
- 用户主动询问"上个月消费分析""消费分析""钱都花哪了"等

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「采纳建议」 | 自动应用推荐预算，保存后展示新预算概览 |
| 点击「自己设置」 | 加载 `references/onboarding.md` §预算调整 |
| 点击「查看趋势图」 | 展示 §近 3 月趋势对比 |
