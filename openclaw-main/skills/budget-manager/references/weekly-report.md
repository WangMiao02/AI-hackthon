# 消费周报 — 卡片模板与消费看板

## 触发条件

- 每周一 9:00 自动推送（`trigger_type == "scheduled"`）
- 用户主动询问"上周花了多少""看看周报""每日明细"等

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「查看每日明细」 | 展示 §消费看板（每日明细） |
| 点击「本月趋势」 | 加载 `references/monthly-report.md` |
| 点击「调整预算」 | 加载 `references/onboarding.md` §预算调整 |

**用户响应分支：**

|------|------|
| 点击「返回周报」 | 返回周报卡片 |

| 按类目筛选（如"只看外卖"） | 过滤仅展示对应类目条目 |
