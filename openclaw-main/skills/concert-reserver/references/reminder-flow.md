# 开票提醒 — 卡片模板与流程细节

## 开售前一天提醒卡片（20:00 推送）

```
┌──────────────────────────────────────┐
│ 🔔 你关注的演出明天开售！             │
│                                      │
│ 🎤 周杰伦「嘉年华」北京站            │
│ 🕙 开售时间：明天 6月1日 10:00       │
│                                      │
│ 你的抢票配置：                       │
│ · 目标票档：内场 ¥1280               │
│ · 备选票档：看台 ¥880                │
│ · 数量：2 张                         │
│ · 实名人：张思怡、李明               │
│                                      │
│ [确认抢票配置] [修改配置] [取消提醒] │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「确认抢票配置」 | 锁定配置，开售时自动执行 §5 抢票 |
| 点击「修改配置」 | 进入抢票配置修改（票档/数量/实名人） |
| 点击「取消提醒」 | 从关注列表移除该演出提醒 |

## 开售倒计时提醒（开售前 30 分钟）

```
┌──────────────────────────────────────┐
│ ⏰ 还有 30 分钟开售！                │
│                                      │
│ 🎤 周杰伦「嘉年华」北京站            │
│ 🕙 10:00 准时开抢                    │
│                                      │
│ ✅ 抢票配置已就绪                    │
│ · 内场 ¥1280 × 2张                  │
│ · 实名人已填写                       │
│                                      │
│ 📌 开售后我会第一时间帮你抢          │
│                                      │
│ [好的，帮我抢] [我自己来]            │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「好的，帮我抢」 | 开售时刻自动执行 §5 抢票 |
| 点击「我自己来」 | 跳转猫眼演出购票页，Skill 结束 |
