# 周期复约与打卡提醒 — 卡片模板

## 健身打卡提醒卡片（定时推送）

到期前一天推送，提醒用户明天该去健身了：

```
┌──────────────────────────────────────┐
│ 🏃 明天该去健身啦！                  │
│                                      │
│ 你已经坚持运动 [N] 次了，继续加油！  │
│                                      │
│ 上次：[上次运动类型] · [上次门店]    │
│ 上次教练：[教练姓名]（如有）         │
│                                      │
│ 要帮你预约明天的课吗？               │
│                                      │
│ [帮我约一下]  [我自己来]  [跳过这次] │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「帮我约一下」 | 直接进入 §3 门店查询，优先复用上次门店+教练+课程 |
| 点击「我自己来」 | 回应「好的，加油！」，Skill 结束 |
| 点击「跳过这次」 | 询问「好的，要调整提醒频率吗？」，可重新设置周期 |

---

## 快捷复约卡片（上次信息完整时）

若上次预约信息完整（门店+教练/班次+时长），展示快捷复约卡片，减少用户操作步骤：

```
┌──────────────────────────────────────┐
│ 🏃 明天该去健身啦！                  │
│                                      │
│ 上次预约：                           │
│ 🏋️ 动感单车（45min）                 │
│ 🏠 超级猩猩（三里屯店）              │
│ 👤 陈教练  ⭐ 4.9分                  │
│                                      │
│ 帮你查了一下，明天这些时段可约：     │
│   🟢 18:30   🟢 20:00               │
│                                      │
│ [约18:30]  [约20:00]  [换个课程]     │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击具体时段 | 直接展示预约确认卡片（跳过门店/教练选择步骤） |
| 点击「换个课程」 | 进入 §2.2 意图识别，重新选择运动类型 |
| 不响应（超时） | 静默，不再推送，等待下次周期 |

## 连续打卡激励卡片

用户连续完成 N 次健身打卡时，在预约成功卡片底部追加激励文案：

| 连续次数 | 激励文案 |
|---------|---------|
| 3次 | 「已连续健身 3 次，习惯正在养成！」 |
| 7次 | 「坚持一周了！你真的很厉害！」 |
| 30次 | 「连续打卡 30 次，你已经是健身达人了！」 |

```
┌──────────────────────────────────────┐
│ ✅ 预约成功！                        │
│ ...（预约详情）...                   │
│                                      │
│ ──────────────────────────────────── │
│ 🔥 已连续健身 7 次，坚持一周了！     │
│    你真的很厉害！继续保持～          │
└──────────────────────────────────────┘
```

## 周期调整卡片

用户主动说「调整提醒频率」或跳过后触发：

```
┌──────────────────────────────────────┐
│ 🔄 调整健身提醒频率                  │
│                                      │
│ 当前设置：每周 2 次                  │
│                                      │
│ [每周1次]  [每周2次]  [每周3次]      │
│ [暂停提醒]  [自定义]                 │
└──────────────────────────────────────┘
```

| 操作 | 处理 |
|------|------|
| 选择新频率 | 更新提醒周期，回应「已更新，下次提醒在 [日期]」 |
| 点击「暂停提醒」 | 暂停周期提醒，可随时重新开启 |
