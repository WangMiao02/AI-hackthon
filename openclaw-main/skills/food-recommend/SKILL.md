# food_recommend — 外卖推荐

## 1. 执行入口判断

**首先判断触发来源，然后跳转对应分支：**

| 来源 | 判断方式 | 跳转 |
|------|---------|------|
| 定时系统触发 | `trigger_type == "scheduled"`（每日 11:00） | → §2 定位获取 |
| 用户主动唤起 | 对话中含外卖/推荐/点餐等意图 | → §2 定位获取 |

---

## 2. 定位获取

1. 尝试获取用户实时 GPS 定位
2. 如果**GPS 授权正常** → 以实时位置为中心，查询 5km 内餐厅
3. 如果**GPS 未授权** → 降级使用用户常用地址，查询 5km 内餐厅
4. 跳转 §3 餐厅筛选

---

## 3. 餐厅筛选与推荐

**输入**：定位坐标 + 用户历史订单 + 用户历史搜索词

**筛选规则**：

| 优先级 | 维度 | 说明 |
|--------|------|------|
| 1 | 历史订单关联性 | 与用户近期高频消费餐厅 / 菜系匹配 |
| 2 | 历史搜索词关联性 | 与用户近期搜索的菜系 / 关键词匹配 |
| 3 | 配送时长 | 同等条件下优先配送更快的餐厅 |

**冷启动处理**（无历史订单和搜索记录）：
→ 直接推送用户定位5km内的热门餐厅，按销量倒序排列

筛选结果取前 **3 条** 展示，超出折叠；如**无匹配餐厅** → 提示「附近暂无推荐餐厅」，Skill 结束

跳转 §4 推送推荐卡片

---

## 4. 推送卡片与确认下单

```
│ 🍱 中午吃什么？为你推荐附近餐厅      │
│                                      │
│ 🥇 老乡鸡（静安寺店）               │
│    均价 ¥25 · 配送约 25 分钟         │
│ 🥈 夸父炸串                          │
│    均价 ¥32 · 配送约 30 分钟         │
│ 🥉 遇见小面                          │
│    均价 ¥28 · 配送约 35 分钟         │
│    + 2家更多推荐 ∨                   │
│                                      │
│  [选第一家] [选第二家] [选第三家]    │
│         [暂时不定]                   │

```

**用户响应分支**：
- 点击 `选第X家` → §4.2 菜品确认页
- 点击 `暂时不定` → Skill 结束，不再重复推送

### 4.2 菜品确认页（对话内展示）

展示该店用户历史最常点的菜品，冷启动时展示该店销量最高的菜品。

```
┌──────────────────────────────────────┐
│ 🍜 老乡鸡（静安寺店）               │
│                                      │
│ 黄焖鸡米饭（大份）         ¥28.00   │
│ 你上次也点了这个 👍                  │
│                                      │
│ 配送时长：约 25 分钟                 │
│ 配送费：¥5.00                        │
│ 合计：¥33.00                         │
│                                      │
│      [确认下单]     [换一家]         │
└──────────────────────────────────────┘
```

**用户响应分支**：
- 点击 `确认下单` → 传入订单信息，触发 `skill_order`，Skill 结束
- 点击 `换一家` → 返回 §4.1 推荐卡片

---

## 5. 边界处理速查

| 场景 | 处理 |
|------|------|
| GPS 未授权 | 降级使用常用地址，卡片注明「基于常用地址推荐」 |
| 常用地址也未设置 | 提示用户设置地址后重试，Skill 结束 |
| 5km 内无匹配餐厅 | 提示「附近暂无推荐餐厅」，Skill 结束 |
| 冷启动（无历史记录） | 推热门餐厅，按销量倒序排列 |
| 用户点击「暂时不定」 | 当日不再推送，次日正常恢复 |
| 所选餐厅已打烊 | 提示「该餐厅暂时不营业」，返回推荐卡片重选 |

## 6. 本期不做（Backlog）

> 以下功能**不在当前 Skill 范围内**，遇到请告知用户暂不支持：

- 自定义推送时间（现阶段固定 11:00）
- 多人拼单
- 用户手动修改推荐菜品
- 晚餐推荐（现阶段仅午餐）
- 跨平台外卖聚合（仅美团）
