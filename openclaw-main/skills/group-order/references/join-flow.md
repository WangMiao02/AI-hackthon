# 加入拼单 — 详细流程

## 2.1 拼单信息展示

从拼单链接解析出发起人和餐厅信息，推送提醒卡片：

```
┌──────────────────────────────────────┐
│ 🍽 同事小王发起了拼单！              │
│                                      │
│ 餐厅：老乡鸡（静安寺店）            │
│ 当前已有 2 人点单                    │
│ 截止时间：11:30（还剩 25 分钟）      │
│                                      │
│      [去选餐]      [不参加]          │
└──────────────────────────────────────┘
```

**用户响应分支**：
- 点击 `去选餐` → §2.2 菜品选择
- 点击 `不参加` → Skill 结束

---

## 2.2 菜品选择

**输入**：餐厅菜品列表 + 用户历史订单

**排序规则**：

| 优先级 | 维度 | 说明 |
|--------|------|------|
| 1 | 用户历史常点 | 该店中用户曾下单的菜品置顶 |
| 2 | 店铺热销 | 该店销量最高的菜品次之 |

**冷启动处理**（该店无历史订单）：
→ 直接展示店铺热销菜品

展示卡片：

```
┌──────────────────────────────────────┐
│ 🛒 选择你要点的菜品                  │
│ 老乡鸡（静安寺店）                   │
│                                      │
│ ⭐ 你常点：                          │
│   黄焖鸡米饭（大份）       ¥28.00   │
│   农家小炒肉               ¥22.00   │
│                                      │
│ 🔥 店铺热销：                        │
│   香菇滑鸡饭               ¥24.00   │
│   番茄蛋花汤               ¥ 8.00   │
│                                      │
│   [确认选品]    [查看完整菜单]       │
└──────────────────────────────────────┘
```

**用户响应分支**：
- 用户选好菜品 + 点击 `确认选品` → §2.3 等待汇总
- 点击 `查看完整菜单` → 展开该店全部菜品列表

---

## 2.3 等待汇总

```
✅ 已提交你的选品！

· 黄焖鸡米饭（大份）  ¥28.00

等待发起人确认下单，届时会通知你。
```

→ 用户端 Skill 结束，等待发起人触发下单
