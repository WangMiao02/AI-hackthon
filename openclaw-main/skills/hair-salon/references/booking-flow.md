```
┌──────────────────────────────────────┐
│ ✂️ 为你找到以下美发门店              │

│    ⭐ 4.8分 · 1056 人消费过          │
│    💰 人均 ¥158 · 距你 1.5km         │
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 🔁 你上次来过这家 | 历史有消费记录时显示，无则不显示 |
| 人均价格 | 取该品类的平均消费金额 |
| 自己看看 | 跳转美团丽人搜索页，Skill 结束 |

## 发型师列表卡片（含作品图）

```
┌──────────────────────────────────────┐
│ 👤 选择发型师                        │
│ TONY'S HAIR STUDIO                   │
│                                      │
│ 🥇 Tony（店长）                      │
│    ⭐ 4.9分 · 服务 1286 次           │
│    🔁 上次为你服务过                 │
│    作品：[图1] [图2] [图3]           │
│                                      │
│ 🥈 小K                               │
│    ⭐ 4.8分 · 服务 632 次            │
│    作品：[图1] [图2] [图3]           │
│                                      │
│ [选Tony] [选小K] [不指定，随机]      │
└──────────────────────────────────────┘
```

**作品图加载规则：**

| 规则 | 说明 |
|------|------|
| 来源 | 从该发型师评论区/作品集拉取最近 3 张代表作品图 |
| 优先级 | 优先展示与本次服务类型匹配的作品（染发→染发作品，烫发→烫发作品） |
| 降级 | 图片加载失败时展示文字标签「擅长：[剪发/烫发/染发/护理]」 |
| 点击图片 | 放大查看，底部显示该作品对应的评论原文 |

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「选XX」 | 展示项目列表卡片（见下） |
| 点击「不指定，随机」 | 跳过发型师选择，展示所有可用时段 |
| 点击作品图放大 | 展示大图 + 评论原文，用户可返回继续选择 |
