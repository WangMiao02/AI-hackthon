# 预约流程 — 卡片模板与交互细节

## 师傅推荐卡片

```
┌──────────────────────────────────────┐
│ 🧹 为你找到以下保洁师傅              │
│ 📍 服务地址：朝阳区XX路XX号          │
│                                      │
│ 🥇 王师傅                            │
│    ⭐ 4.9分 · 服务 386 次            │
│    💰 ¥68/小时 · 最快明天上门        │
│    🔁 曾服务过你家                   │
│                                      │
│ 🥈 李师傅                            │
│    ⭐ 4.8分 · 服务 214 次            │
│    💰 ¥65/小时 · 最快明天上门        │
│    + 3位更多师傅 ∨                   │
│                                      │
│ [选王师傅]  [选李师傅]  [自己看看]   │
└──────────────────────────────────────┘
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 🔁 曾服务过你家 | 历史有服务记录时显示，无则不显示 |
| 最快上门 | 取该师傅最近可用时段的日期 |
| 自己看看 | 跳转美团家政搜索页，Skill 结束 |

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「选XX师傅」 | 展示时间窗口选择卡片（见下） |
| 说"第一个"/"选王师傅" | 识别后跳转时间窗口选择 |
| 点击「自己看看」 | 跳转美团家政搜索页，Skill 结束 |

**用户响应分支：**

## 维修类附加：故障描述输入

维修意图识别后，在展示师傅推荐卡片前先追问：

```
🔧 方便描述一下是什么问题吗？
（例如：马桶堵了、灯不亮了、水管漏水）

[跳过，直接找师傅]
```

用户输入故障描述后，将描述传入查询接口匹配专项师傅（如水电工、家电维修工）；点击「跳过」则展示通用维修师傅列表。
