# 时间收集流程 — 卡片模板与交互细节

## 时间收集预览卡片（发起人端）

发起人确认信息后，展示预览供最终确认：

```
┌──────────────────────────────────────┐
│ 📅 时间收集预览                      │
│                                      │
│ 🎯 周末聚餐                          │
│ 👥 参与人：小王、小李、小张 + 你     │
│ 📆 候选范围：5月31日（周六）~        │
│             6月1日（周日）           │
│ ⏰ 时间段：17:00 - 22:00            │
│ ⏳ 截止：明天 12:00 自动汇总         │
│                                      │
│ 确认后我会生成链接，你发给他们填就行~ │
│                                      │
│    [确认，生成链接]    [我要改一下]   │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「确认，生成链接」 | 调用时间收集链接生成接口 → 展示分享卡片 |
| 点击「我要改一下」 | 展示可修改项（日期范围/时间段/参与人/截止时间） |

**修改交互**：

```
你想改哪个？

[改日期范围]  [改时间段]  [加/减人]  [改截止时间]
```

## 分享链接卡片（发起人端）

```
┌──────────────────────────────────────┐
│ ✅ 时间收集已创建！                  │
│                                      │
│ 📅 周末聚餐 — 大家来填时间           │
│ 👥 等待：小王、小李、小张            │
│ 📆 5月31日~6月1日 17:00-22:00       │
│ ⏰ 截止：明天 12:00 自动汇总         │
│                                      │
│ 🔗 https://meituan.com/meetup/xxxxx  │
│                                      │
│ [复制链接]  [查看填写情况]           │
│                                      │
│ 📌 朋友点击链接标记有空的时段即可，  │
│    无需下载 App。                    │
│                                      │
│ 💡 你自己也记得标一下有空的时间哦~   │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「复制链接」 | 复制链接到剪贴板，提示「链接已复制，发给朋友吧！」 |
| 点击「查看填写情况」 | 展示填写进度卡片（见下） |
| 说「填得怎么样了」 | 同上，展示填写进度卡片 |

## 朋友端：时间填写页面（H5）

朋友点击分享链接后看到的页面：

```
┌──────────────────────────────────────┐
│ 📅 周末聚餐 — 大家来填时间           │
│ 发起人：Sarah | 已有 2 人填写        │
│                                      │
│ 请标记你有空的时段（可多选）：       │
│                                      │
│          5/31 周六    6/1 周日       │
│ 17:00    ☐           ☐             │
│ 17:30    ☐           ☐             │
│ 18:00    ☐           ☐             │
│ 18:30    ☐           ☐             │
│ 19:00    ☐           ☐             │
│ 19:30    ☐           ☐             │
│ 20:00    ☐           ☐             │
│ 20:30    ☐           ☐             │
│ 21:00    ☐           ☐             │
│ 21:30    ☐           ☐             │
│                                      │
│ 你的昵称：[________]                 │
│                                      │
│         [提交我的时间]               │
└──────────────────────────────────────┘
```

**填写规则：**

| 规则 | 说明 |
|------|------|
| 时间粒度 | 30 分钟为一格 |
| 最少选择 | 至少标记 1 个时段 |
| 昵称 | 如果发起人已填写参与人昵称，自动匹配；否则需手动输入 |
| 重复提交 | 允许修改（覆盖上次填写） |

**提交成功后**：

```
✅ 已收到你的时间！

你标记了 5 个可用时段。
等其他人也填完后，发起人会通知大家最终时间~

目前填写情况：
✅ Sarah — 已填写
✅ 小王 — 已填写
✅ 你 — 刚刚提交
⏳ 小张 — 还没填
```

## 填写进度卡片（发起人端）

发起人查看进度时展示：

```
┌──────────────────────────────────────┐
│ 📊 填写进度                          │
│                                      │
│ 📅 周末聚餐                          │
│ 创建于 2 小时前 | 截止：明天 12:00   │
│                                      │
│ ✅ Sarah（你）— 已填写               │
│ ✅ 小王 — 已填写（30分钟前）         │
│ ✅ 小李 — 已填写（10分钟前）         │
│ ⏳ 小张 — 还没填                     │
│                                      │
│ [催一下小张]  [不等了，先看结果]     │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「催一下小张」 | 生成催促文案（见下） |
| 点击「不等了，先看结果」 | 基于已有 3 人数据计算交集 → 跳转结果卡片 |

## 催促文案（供用户复制）

```
┌──────────────────────────────────────┐
│ 📋 催促文案已复制到剪贴板：          │
│                                      │
│ "小张～帮忙填一下时间呀，就点几下    │
│  的事儿：https://meituan.com/meetup/ │
│  xxxxx                               │
│  大家等你定时间呢！"                 │
│                                      │
│ [换一个说法]  [好的，我去发]         │
└──────────────────────────────────────┘
```

点击「换一个说法」→ 重新生成一条不同风格的催促文案：

```
"时间填写链接还没填呢～花 10 秒标一下
 你有空的时段就行：[链接]
 不然我们就不等你啦 😂"
```

## 全员填写完成 — 自动推送通知

所有参与人填写完毕后，自动推送给发起人：

```
┌──────────────────────────────────────┐
│ 🔔 所有人都填完啦！                  │
│                                      │
│ 📅 周末聚餐 — 4/4 人已填写          │
│                                      │
│ 我来帮你看看大家什么时候都有空~      │
│                                      │
│ [查看结果]                           │
└──────────────────────────────────────┘
```

→ 点击「查看结果」→ 跳转 result-flow.md 中的结果卡片

---

## 截止时间到达 — 自动汇总通知

截止时间到但仍有人未填写时：

```
┌──────────────────────────────────────┐
│ ⏰ 时间收集已截止                    │
│                                      │
│ 📅 周末聚餐                          │
│ 已填写：3/4 人（小张未填写）         │
│                                      │
│ 我先基于已有的 3 人数据帮你算一下~   │
│                                      │
│ [查看结果]  [再等等，延长截止时间]   │
└──────────────────────────────────────┘
```

| 操作 | 处理 |
|------|------|
| 点击「查看结果」 | 基于已有数据计算交集 → 跳转结果卡片 |
| 点击「再等等，延长截止时间」 | 截止时间延长 12 小时 |
