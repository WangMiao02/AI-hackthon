## 天气区块（恶劣天气版）

当检测到雨天/高温/大风/雾霾/雪天时，天气区块底部追加联动提示：

```
┌──────────────────────────────────────┐
│ 🌧 北京 · 中雨                       │
│ 🌡 16°C ~ 22°C  💨 北风 4级          │
│ 未来3天：周三🌧20° · 周四☁️24° · 周五☀️27° │
│                                      │
│ 🌂 今天有雨，出门记得带伞            │
│ 今天天气不好，需要帮你叫车吗？       │
│ [帮我叫车]  [不用了]                 │
└──────────────────────────────────────┘
```

**恶劣天气判断标准：**

| 天气类型 | 触发条件 |
|---------|---------|
| 雨天 | 小雨及以上 |
| 高温 | 最高气温 ≥ 35°C |
| 大风 | 风力 ≥ 5级 |
| 雾霾 | AQI ≥ 150 |
| 雪天 | 小雪及以上 |

## 联动按钮行为

| 按钮 | 触发 Skill | 传入参数 |
|------|-----------|---------|
| 🍳 帮我订早餐 | `food_recommend` | `meal_type = "breakfast"`，使用用户当前位置 |
| 🚗 帮我叫车（行程条目） | 先询问「你打算几点出发？」 → 用户填写后触发 `ride_hailing` | `destination = 预约地点`，`departure_time = 用户填写时间` |
| 🚗 帮我叫车（底部按钮） | `ride_hailing` | 使用用户当前位置，目的地由用户在 ride_hailing 中填写 |

## 今日行程区块（有预约时展示）

```
┌──────────────────────────────────────┐
│ 📅 今日行程                          │
│ · 10:00  美甲  小雅（朝阳大悦城）   │
│   [帮我叫车]                         │
│ · 19:30  餐厅  外婆家（三里屯店）   │
│   [帮我叫车]                         │
└──────────────────────────────────────┘
```

点击「帮我叫车」后弹出询问：

```
你打算几点出发去 小雅（朝阳大悦城）？
[输入出发时间，如 09:30]
[确认]  [取消]
```

用户确认后触发 `ride_hailing`，传入目的地和出发时间。
