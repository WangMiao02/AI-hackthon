## 购票确认卡片

```
┌──────────────────────────────────────┐
│ 🎫 确认购票                          │
│                                      │
│ 🎬 流浪地球3                         │
│ 📍 万达影城（中关村店）              │
│ 📅 5月25日（周六）10:30  IMAX        │
│ 💺 D排 5、6号                        │
│ 💰 ¥79 × 2 = ¥158.00                │
│ 支付方式：美团支付（默认）           │
│                                      │
│      [确认购票]     [重新选座]       │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「确认购票」 | 调用 `order` skill 完成支付 → 购票结果处理 |
| 点击「重新选座」 | 返回选座卡片 |
