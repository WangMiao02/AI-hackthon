> **动态标签说明**：风格分组标签由门店实际作品标签动态生成，不固定枚举。若用户有指定款式关键词（如"海莉甲"），优先将该标签置顶展示。

**交互说明：**

| 操作 | 处理 |
|------|------|
| 点击某张图片 | 放大展示 + 显示评论原文（含用户评价和技师备注） |
| 点击「查看更多」 | 展开该风格下更多款式图 |
| 点击「直接预约，不选款式」 | 跳过款式选择，进入技师选择卡片 |
| 在大图页点击「就要这款」 | 记录款式偏好，进入技师选择卡片（技师备注中附带款式图链接） |
