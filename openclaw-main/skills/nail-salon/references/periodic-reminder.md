# 周期复约提醒 — 卡片模板与交互细节

## 到期复约提醒卡片

到期前一天推送：

```
┌──────────────────────────────────────┐
│ 🔔 美甲复约提醒                      │
│                                      │
│ 距离你上次做「光疗甲」已过去 3 周，  │
│ 该换新款式啦～                       │
│                                      │
│ 上次信息：                           │
│ 🏠 指尖美学                          │
│ 👤 小雅  ⭐ 4.9分                    │
│ 💰 ¥168                              │
│                                      │
│ [一键复约（同门店同技师）]           │
│ [看看新款式]  [下次再说]  [取消提醒] │
└──────────────────────────────────────┘
```

## 用户响应分支

| 操作 | 处理 |
|------|------|
| 点击「一键复约」 | 使用上次门店+技师信息，直接跳转时段选择卡片 |
| 点击「看看新款式」 | 跳转至该门店最新款式图集页（§4 款式图集），用户选款后继续预约流程 |
| 点击「下次再说」 | 将提醒推迟 3 天后再次推送（仅推迟一次，第二次仍不操作则本轮静默） |
| 点击「取消提醒」 | 删除该周期提醒任务，不再推送 |

## 推迟逻辑

- 第一次「下次再说」→ 3 天后再推送一次
- 第二次仍无操作 → 本轮静默，等待下一个完整周期后再提醒
- 用户主动预约后 → 以新预约日期重新计算下次提醒时间

## 复约信息更新

当用户通过复约完成新预约时，自动更新记录：
- 更新上次门店/技师/项目信息（如有变化）
- 重新计算下次提醒时间 = 新预约日期 + 用户设定周期 - 1天
