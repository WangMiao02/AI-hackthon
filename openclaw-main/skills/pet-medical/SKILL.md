# pet_medical — 宠物医疗

## 1. 执行入口判断

| 来源 | 判断方式 | 跳转 |
|------|---------|------|
| 定时系统触发（疫苗/驱虫到期） | `trigger_type == "scheduled"` | → §4 计划性提醒 |
| 用户主动-计划性 | 含打疫苗/驱虫/体检意图 | → §3 计划性预约 |
| 用户主动-突发性 | 含生病/受伤/不吃东西/异常行为 | → §2 突发就诊 |
| 用户无宠物档案 | memory 中无宠物记录 | → 调用 pet-grooming §6 建档 |

---

## 2. 突发就诊

**设计原则**：快速给信息，不走复杂预约流程。

**执行步骤**：

1. 简单了解情况（不做诊断）：「[宠物名]怎么了？是不吃东西、精神不好、还是有外伤？」
2. 根据紧急程度分流：
   - 紧急关键词（抽搐/大出血/中毒/呼吸困难）→ 直接推送 24 小时急诊医院
   - 非紧急 → 推送附近评分最高的宠物医院

3. 推送就近医院卡片：

```
┌──────────────────────────────────────┐
│ 🏥 附近宠物医院                      │
│                                      │
│ 🥇 瑞派宠物医院（朝阳店）           │
│    ⭐ 4.9分 · 距你 1.3km            │
│    🕐 营业中（24小时）               │
│    📞 010-XXXXXXXX                   │
│                                      │
│ 🥈 芭比堂动物医院                   │
│    ⭐ 4.8分 · 距你 2.1km            │
│    🕐 营业中（09:00-21:00）          │
│    📞 010-XXXXXXXX                   │
│                                      │
│ [导航去第一家]  [拨打电话]           │
│ [帮我叫车过去]                       │
└──────────────────────────────────────┘
```

4. 用户选择「帮我叫车」→ 联动 `ride_hailing`
5. 就诊结束后（用户反馈或次日）→ 询问就诊结果，更新档案，联动 §5

---

## 3. 计划性预约（用户主动）

1. 确认宠物（多宠时询问）
2. 确认服务类型（疫苗/驱虫/体检）
3. 推荐附近宠物医院（排序：历史去过 > 评分 > 距离）
4. 用户选医院 → 展示可选时段
5. 用户选时段 → 确认预约 → 调用 `order` 支付

### 3.1 预约成功

```
✅ 预约成功！
🐾 [宠物名] · [服务类型]
🏥 [医院名称]
📅 [日期] [时间]
📍 [地址]
💰 ¥[金额]
📌 就诊当天会提醒你出发。
```

→ 就诊当天早上提醒 + 联动 `ride_hailing` 建议叫车
→ 将本次记录写入 memory（疫苗类型、日期），自动计算下次到期时间

---

## 4. 计划性提醒（定时触发）

根据 memory 中记录的上次疫苗/驱虫日期 + 标准周期计算到期日，提前 7 天推送：

```
┌──────────────────────────────────────┐
│ 🔔 健康提醒                          │
│                                      │
│ [宠物名]的[疫苗类型]将于[日期]到期   │
│ 建议提前预约接种～                    │
│                                      │
│ [帮我预约]  [知道了，稍后处理]       │
└──────────────────────────────────────┘
```

| 操作 | 处理 |
|------|------|
| 帮我预约 | 跳转 §3 计划性预约 |
| 稍后处理 | 到期前 1 天再提醒一次 |

---

## 5. 就诊后联动

就诊结束后询问：「[宠物名]看完医生了吗？医生有开什么药或特殊饮食建议吗？」

根据用户反馈：
- 开了药/营养品 → 联动 `pet-shop`：「要帮你买吗？」
- 需要长期服药 → 联动 `pet-supplies`：「要加入定期补买提醒吗？」
- 建议换粮 → 联动 `pet-shop` 推荐处方粮
- 建议药浴 → 联动 `pet-grooming` 提醒可预约药浴

---

## 6. 边界处理速查

| 场景 | 处理 |
|------|------|
| 紧急情况 | 直接给 24 小时急诊医院，不走预约 |
| 用户在外地（如出差中宠物在寄养） | 推荐寄养门店附近的医院 |
| 疫苗种类不确定 | 提示常见疫苗周期供参考，建议咨询医生 |
| 附近无宠物医院 | 扩大搜索范围 |
| 用户问诊断建议 | 明确告知不提供诊断，建议到院检查 |

## 7. 疫苗/驱虫标准周期参考

| 项目 | 标准周期 | 说明 |
|------|---------|------|
| 猫三联 | 首年3针，之后每年1针 | 幼猫8周起 |
| 狂犬疫苗 | 每年1针 | 猫犬通用 |
| 体内驱虫 | 每3个月 | 室内猫可6个月 |
| 体外驱虫 | 每月 | 外出频繁的犬 |
| 年度体检 | 每年1次 | 7岁以上建议半年1次 |

> 以上仅为提醒参考，实际周期以医生建议为准。
