# 发起投票 — 卡片模板与交互细节

## 候选项挑选卡片（酒店示例）

```
┌──────────────────────────────────────┐
│ 🏨 选择放入投票的酒店（2–6项）       │
│                                      │
│ ☑ 🥇 静安香格里拉                   │
│      ¥580/晚 · 距办公地 800m         │
│      ⭐ 4.9分                        │
│                                      │
│ ☑ 🥈 CitiGO欢阁酒店                 │
│      ¥320/晚 · 距办公地 1.2km        │
│      ⭐ 4.7分                        │
│                                      │
│ ☐ 🥉 全季酒店（静安店）             │
│      ¥280/晚 · 距办公地 2.1km        │
│      ⭐ 4.6分                        │
│                                      │
│ [+ 手动添加美团链接]                 │
│                                      │
│ 已选 2 项                            │
│      [下一步：设置投票]              │
└──────────────────────────────────────┘
```

**候选项挑选规则：**

| 规则 | 说明 |
|------|------|
| 最少 2 项 | 少于 2 项时「下一步」置灰 |
| 最多 6 项 | 超出时提示「最多添加 6 个候选项」 |
| 勾选/取消 | 点击候选项行切换勾选状态 |

## 手动添加美团链接

用户点击「+ 手动添加美团链接」后：

```
┌──────────────────────────────────────┐
│ 📎 粘贴美团商品链接                  │
│                                      │
│ [请粘贴美团酒店/餐厅/外卖链接...]    │
│                                      │
│      [解析链接]     [取消]           │
└──────────────────────────────────────┘
```

解析成功后自动追加到候选项列表：

```
✅ 解析成功，已添加：

☑ 朗廷酒店（上海）
   ¥1,280/晚 · 外滩附近
   ⭐ 4.9分
```

解析失败时：
```
⚠️ 该链接无法解析，请确认是否为美团商品链接。
```

│       [修改标题]                     │
│                                      │
│ 截止时间：不限时  [设置截止时间]     │
│                                      │
│ 显示实时票数：✅ 开启  [关闭]        │
│                                      │
│ 投票方式：单选  [改为多选]           │
│                                      │
│ 候选项预览：                         │
│   1. 静安香格里拉  ¥580/晚           │
│   2. CitiGO欢阁酒店  ¥320/晚         │

**截止时间设置（可选）：**

```
选择截止时间：

[2小时后]  [今晚24:00]  [明天此时]  [自定义]
```

## 投票分享卡片

```
┌──────────────────────────────────────┐
│ ✅ 投票已创建！                      │
│                                      │
│ 📊 长沙住哪？大家来选                │
│                                      │
│ 🏨 静安香格里拉  ¥580/晚            │
│ 🏨 CitiGO欢阁酒店  ¥320/晚          │
│                                      │
│ 截止：不限时 | 当前参与：0 人        │
│                                      │
│ 🔗 https://meituan.com/poll/xxxxxx   │
│                                      │
│ [复制链接]  [查看投票页]  [结束投票] │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「复制链接」 | 复制投票链接到剪贴板，提示「链接已复制，发给朋友吧！」 |
| 点击「查看投票页」 | 跳转投票 H5 页面 |
| 点击「结束投票」 | 停止收票，跳转结果汇总卡片（见 vote-flow.md） |
| 说「看看投票结果」 | 拉取当前票数，展示实时结果卡片 |
