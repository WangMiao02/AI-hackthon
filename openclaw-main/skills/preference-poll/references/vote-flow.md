## 最终结果汇总卡片（投票结束后）

发起人点击「结束投票」或截止时间到期后推送：

```
┌──────────────────────────────────────┐
│ 🎉 投票结束！                        │
│                                      │
│ 📊 长沙住哪？大家来选                │
│ 共 5 人参与                          │
│                                      │
│ 🏆 胜出：静安香格里拉                │
│    ¥580/晚 · ⭐ 4.9分               │
│    3票（60%）                        │
│                                      │
│ ─────────────────────────────────── │
│ 完整票数：                           │
│ 🥇 静安香格里拉      3票  ████████  │
│ 🥈 CitiGO欢阁酒店    2票  █████░░░  │
│ 🥉 朗廷酒店（上海）  0票  ░░░░░░░░  │
│                                      │
│ 要帮你预订静安香格里拉吗？           │
│                                      │
│    [帮我预订]       [我自己来]       │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「帮我预订」 | 触发 `hotel_reserver`，目标酒店预填为胜出选项 |
| 点击「我自己来」 | Skill 结束，不触发预订 |

**平票处理（多个选项票数相同）：**

```
┌──────────────────────────────────────┐
│ 📊 投票结束，出现平票！              │
│                                      │
│ 🤝 静安香格里拉  2票                 │
│ 🤝 CitiGO欢阁    2票                 │
│                                      │
│ 由你来做最终决定：                   │
│                                      │
│ [选静安香格里拉]  [选CitiGO欢阁]     │
└──────────────────────────────────────┘
```

→ 发起人选择后，展示最终结果汇总卡片，并提供预订入口

## 各类投票的预订联动

| 投票类型 | 胜出后联动 skill |
|---------|----------------|
| 酒店投票 | `hotel_reserver`，预填目标酒店 |
| 餐厅/套餐投票 | 跳转美团餐厅页，暂不联动 skill |
| 外卖投票 | `food_recommend`，预填目标餐厅 |
