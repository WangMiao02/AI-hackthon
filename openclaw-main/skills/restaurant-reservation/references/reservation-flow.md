# 订座流程 — 卡片模板与交互细节

## 餐厅推荐卡片

```
┌──────────────────────────────────────┐
│ 🍽 为你找到以下餐厅                  │
│ 明晚 · 4人 · 火锅                    │
│                                      │
│ 🥇 海底捞（朝阳大悦城店）            │
│    ⭐ 4.8分 · 月销 2341 单           │
│    💰 人均 ¥128 · 距你 1.5km         │
│    🟢 支持在线订座                   │
│    🔁 你上次来过这家                 │
│                                      │
│ 🥈 小龙坎（三里屯店）                │
│    ⭐ 4.7分 · 月销 1856 单           │
│    💰 人均 ¥108 · 距你 2.3km         │
│    🟢 支持在线订座                   │
│                                      │
│ 🥉 巴奴毛肚火锅                      │
│    ⭐ 4.9分 · 月销 986 单            │
│    💰 人均 ¥158 · 距你 3.1km         │
│    📞 仅支持电话预约                 │
│    + 2家更多 ∨                       │
│                                      │
│ [选第一家] [选第二家] [选第三家]     │
└──────────────────────────────────────┘
```

**字段说明：**

| 字段 | 说明 |
|------|------|
| 🟢 支持在线订座 | 可直接按钮预约，优先推荐 |
| 📞 仅支持电话预约 | 选择后展示电话预约提示卡片 |
| 🔁 你上次来过这家 | 历史有消费记录时显示 |

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 选择「🟢 支持在线订座」的餐厅 | 展示时段选择卡片（见下） |
| 选择「📞 仅支持电话预约」的餐厅 | 展示电话预约提示卡片（见 §3） |

## 时段选择卡片

```
┌──────────────────────────────────────┐
│ 📅 选择用餐时间                      │
│ 海底捞（朝阳大悦城店）· 4人          │
│                                      │
│ 明天（5月27日 周二）晚餐             │
│   🟢 17:30   🟢 18:00   🔴 18:30    │
│   🟢 19:00   🔴 19:30   🟢 20:00    │
│                                      │
│ 🟢 可预约  🔴 已约满                 │
│                                      │
│ [更多时段 ∨]                         │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击某时段 | 展示订座确认卡片（见下） |
| 点击「更多时段」 | 展开更多可用时段 |
| 所有时段已满 | 提示「该餐厅今晚已约满，是否换一家或换日期？」 |

## 订座确认卡片

```
┌──────────────────────────────────────┐
│ 🗓 确认订座                          │
│                                      │
│ 🍜 海底捞（朝阳大悦城店）            │
│ 📅 5月27日（周二）18:00              │
│ 👥 4 人                              │
│ 📍 朝阳区大悦城 5楼                  │
│ 📞 联系电话：138xxxxxxxx             │
│                                      │
│ ⚠️ 请在预约时间前 15 分钟到店，      │
│    超时可能取消预约。                │
│                                      │
│      [确认订座]     [换个时间]       │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「确认订座」 | 调用在线订座接口 → §5.1 预约结果处理 |
| 点击「换个时间」 | 返回时段选择卡片 |

## 用餐提醒推送（预约成功后 1 小时前触发）

```
┌──────────────────────────────────────┐
│ 🔔 用餐提醒                          │
│                                      │
│ 你在海底捞的订座还有 1 小时          │
│ 📅 今晚 18:00 · 4人                  │
│ 📍 朝阳区大悦城 5楼                  │
│                                      │
│ [帮我叫车]  [已在路上]               │
└──────────────────────────────────────┘
```

**用户响应分支：**

| 操作 | 处理 |
|------|------|
| 点击「帮我叫车」 | 触发 `ride_hailing`，目的地自动填入餐厅地址 |
| 点击「已在路上」 | 关闭提醒，Skill 结束 |
| 无响应 | 静默，不重复推送 |
