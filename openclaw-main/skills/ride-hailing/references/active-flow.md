# 主动叫车 — 卡片模板与交互细节

## 车型选择卡片

```
┌─────────────────────────────────────────┐
│ 🚗 为您推荐车型                           │
│ 📍 出发：[出发地]                         │
│ 📍 目的地：[目的地]                       │
│ ⏱ 预计行程：[时长] | 💰 预估费用：[价格区间] │
├─────────────────────────────────────────┤
│ ⭐ [车型1]  等待约X分钟  ¥XX–XX           │
│    [车型2]  等待约X分钟  ¥XX–XX           │
│    [车型3]  等待约X分钟  ¥XX–XX           │
├─────────────────────────────────────────┤
│  [确认叫车]          [取消]               │
└─────────────────────────────────────────┘
```

**字段说明：**

| 字段 | 数据来源 | 备注 |
|------|---------|------|
| 出发地 | GPS 实时定位 / 常用地址 | GPS 失败时注明「基于常用地址」 |
| 目的地 | 用户输入 / 上游 Skill 传入 | 联动场景预填 |
| 车型排序 | 历史偏好 → 等待时长 → 价格 | 冷启动时按价格从低到高 |
| ⭐ 标记 | 历史最常选车型 | 无历史记录时不显示 |
| 预估费用 | 美团打车预估接口 | 显示区间，非精确值 |

## 叫车成功卡片

```
┌─────────────────────────────────────────┐
│ ✅ 已成功叫车                             │
│ 🚗 [车型] · [车牌号]                      │
│ 👤 司机：[姓名]  ⭐ [评分]               │
│ ⏱ 预计到达：约 X 分钟                    │
│ 📍 当前位置：[司机实时位置描述]            │
├─────────────────────────────────────────┤
│  [查看实时位置]      [联系司机]            │
└─────────────────────────────────────────┘
```

## 交互分支

| 用户操作 | 处理 |
|---------|------|
| 点击「确认叫车」 | 调用叫车接口，展示叫车成功卡片 |
| 点击「取消」 | 关闭卡片，Skill 结束 |
| 叫车接口失败 | 提示失败原因，保留目的地可重试 |
| 附近无车 | 提示「当前附近暂无可用车辆，请稍后重试」 |
